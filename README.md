Simple proof of concept that takes a voice intent and proxies an API call to retrieve data from [Wireless Registry](https://wirelessregistry.com/index.html)

**Commands:**

From the `functions` directory:

Compile once:
```
$ yarn build
```
Hot reload:
```
$ yarn watch
```
Run tests:
```
$ yarn test
```
Deploy to GCF:
```
$ yarn deploy
```
